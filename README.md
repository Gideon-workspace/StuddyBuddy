**StuddyBuddy** is a full-stack web application that helps students track study activities, collaborate with peers, and receive personalized academic support using data-driven insights. The platform combines a friendly social environment with smart analytics to optimize learning outcomes.

---

## Features

- **Personalized Study Tracking:** Monitor your study sessions, exam scores, and attendance.  
- **Friend Activity Feed:** See what your peers are studying across universities.  
- **Real-time Chat:** Ask questions and collaborate with friends.  
- **Data-driven Insights:** Receive analytics to improve study efficiency.  
- **Cross-platform:** Works on web with a responsive design.

---

## Tech Stack

### Frontend
- React + Vite
- Tailwind CSS for styling
- React Router for page navigation
- Context API for state management

### Backend
- Node.js + Express
- RESTful API design
- JWT Authentication
- MongoDB (or your database of choice)

### Optional / Future
- Python ML service for predictive analytics
- CI/CD with GitHub Actions
- Code coverage tracking with Codecov
